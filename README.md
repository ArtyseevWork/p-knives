> ℹ️ **About this repository.** This is a public **demo** version of the codebase,
> shared as a portfolio sample — private keys and credentials are removed.
> The full app described above is the **production** version live on Google Play.

## What it does

- Measures the sharpening angle of a knife in real time using the **accelerometer**
- Stores sharpening history and per-knife information
- Syncs data across devices via the cloud

## Tech & architecture

- **Flutter** (cross-platform mobile)
- Architecture: **Clean Architecture + MVVM**
- **Firebase**: Google Sign-In auth, Cloud Storage (images), Realtime/Cloud Database (sync)
- In-app monetization integrated via **Adapty**
